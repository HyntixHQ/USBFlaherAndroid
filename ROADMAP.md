# Roadmap

Future plans for **USB Flasher for Android**.

## Phase 1: High-Saturation (Current)
- [x] Initial 1.0.0 Release
- [x] Parallel 32-URB Pipeline (Rust)
- [x] 256MB Prefetch Buffering
- [x] Rootless OTG Support
- [x] SHA-256 Verification

## Phase 2: Refinement (Next)
- [ ] Support for **Windows ISOs** (UEFI partitioning and WIM splitting).
- [ ] Support for **macOS DMG** images.
- [ ] Improved Disk Reader performance for slower SD cards.
- [ ] Localization (Multi-language support).

## Phase 3: Advanced Features
- [ ] Direct image download from URL.
- [ ] Support for flashing compressed images (.gz, .xz) on the fly.
- [ ] Advanced verification (Sector-by-sector comparison).
