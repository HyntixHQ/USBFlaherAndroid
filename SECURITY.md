# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the maintainers. All security vulnerabilities will be promptly addressed.

Please include the following in your report:
- Type of issue (e.g., memory corruption in the Rust engine, credential leak in logs, etc.)
- Steps to reproduce the issue.
- Any potential impact of the vulnerability.

We appreciate your help in keeping this project and its users safe!
